# Notebook 学习指南

10 个 Notebook 按顺序学习，每个 30-60 分钟。

## 学习路径

```
┌─────────────────────────────────────────────────────┐
│  基础概念（01-03）                                  │
│  01 目标检测 → 02 数据集 → 03 架构                  │
├─────────────────────────────────────────────────────┤
│  动手实践（04-06）                                  │
│  04 推理 → 05 训练 → 06 迁移学习                    │
├─────────────────────────────────────────────────────┤
│  进阶技能（07-10）                                  │
│  07 评估 → 08 增强 → 09 自定义数据 → 10 部署        │
└─────────────────────────────────────────────────────┘
```

## Notebook 索引

| # | Notebook | 时长 | 前置知识 | 核心内容 |
|---|----------|------|----------|----------|
| 01 | 什么是目标检测 | 20min | 无 | 检测 vs 分类、YOLO 简介 |
| 02 | 数据集与标注 | 30min | 01 | YOLO TXT 格式、data.yaml |
| 03 | YOLO 架构 | 40min | 01 | Backbone-Neck-Head、FPN |
| 04 | 用预训练模型推理 | 30min | 01,03 | 前处理→推理→后处理 |
| 05 | 训练基础 | 45min | 02,03 | Loss、学习率、超参数 |
| 06 | 迁移学习 | 40min | 05 | 微调、冻结策略 |
| 07 | 评估指标 | 35min | 04 | IoU、mAP、PR 曲线 |
| 08 | 数据增强 | 30min | 05 | Mosaic、MixUp、增强对比 |
| 09 | 自定义数据集 | 45min | 02,05 | 标注、格式转换、数据集划分 |
| 10 | 导出与部署 | 30min | 04 | ONNX、CoreML、量化 |

**总时长：约 5-6 小时**

## 每个 Notebook 的结构

```
📌 学习目标        — 本章要掌握什么
🔗 前置知识        — 需要先学哪些章节
📖 正文 + 教学演示 — 核心内容
💡 类比解释        — 用直觉理解概念
🔍 检查点          — 自测题（pedagogy.checkpoint）
✍️ 练习            — 分层练习（pedagogy.scaffold）
❌ 常见错误        — 避坑指南（pedagogy.reflection）
🤔 反思问题        — 自我评估
📝 自评量表        — 追踪学习进度
🔗 下节预告        — 接下来学什么
```

## 学习建议

1. **按顺序学习** — 每章依赖前面的知识
2. **先看再跑** — 看懂代码再运行，预测输出
3. **做练习** — 只看不练等于没学
4. **记笔记** — 遇到的问题和解决方法记下来
5. **完成后做实战** — 试试 [practice/](../practice/) 的安全帽检测挑战

## 快速命令

```bash
# 启动 Jupyter
uv run jupyter notebook

# 只安装 notebook 依赖
uv sync --extra notebook

# 运行特定 notebook
uv run jupyter nbconvert --execute notebooks/05_training_basics.ipynb
```
