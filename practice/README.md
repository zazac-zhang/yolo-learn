# 🏗️ 实战挑战：安全帽检测

> 学完了 10 个 Notebook？现在来真格的。

## 背景

建筑工地上，工人必须佩戴安全帽和反光衣。用 YOLO 训练一个检测模型，识别以下 3 类目标：

| 类别 | 英文 | 说明 |
|------|------|------|
| 人 | person | 工人 |
| 安全帽 | helmet | 头部防护装备 |
| 反光衣 | vest | 高可见度背心 |

## 数据集特点

- **~500 张图片**，来自真实工地场景
- **3 个类别**，但分布不均匀（人多，安全帽/反光衣少）
- **YOLO TXT 格式**，直接可用
- 来源：[Roboflow Universe](https://universe.roboflow.com/)

### 挑战点

1. **类不平衡**：`person` 数量远多于 `helmet` 和 `vest`
2. **遮挡**：安全帽可能被人头遮挡
3. **小目标**：远景图片中目标很小
4. **相似外观**：安全帽和普通帽子容易混淆

## 任务清单

### 必做（及格线）

- [ ] 下载数据集
- [ ] 分析数据集（类分布、标注质量、图片尺寸分布）
- [ ] 配置 `data.yaml`
- [ ] 训练模型（至少 20 轮）
- [ ] 评估模型，输出 mAP@0.5
- [ ] 分析各类别 AP，找出最差的类别

### 进阶（良好）

- [ ] 分析最差类别 mAP 低的原因（看混淆矩阵、错误样本）
- [ ] 尝试数据增强改进（mixup、更强的 mosaic 等）
- [ ] 对比不同模型大小（n vs s）的效果

### 挑战（优秀）

- [ ] mAP@0.5 > 0.65
- [ ] 写出完整的诊断报告（问题→原因→改进→效果）
- [ ] 尝试冻结 Backbone 微调 vs 全部训练的对比

## Roboflow API Key

1. 注册 [roboflow.com](https://roboflow.com/)（免费）
2. 进入 Settings → API → 获取 API Key
3. 运行 `download.py` 时输入 API Key

或设置环境变量：
```bash
export ROBOFLOW_API_KEY="your_key_here"
```
